# Sber Investment Advisor

Интерактивный дашборд финансового инвестиционного советника для клиентов Сбера.

## Возможности

- 🧠 Профилирование инвестора (3 профиля: консервативный, умеренный, агрессивный)
- 📊 Аллокация активов с интерактивной Doughnut-диаграммой
- 🏦 Рекомендуемые продукты Сбера под каждый профиль
- 📈 Динамика портфеля vs бенчмарк (IMOEX)
- 🌐 Рынки в реальном времени (симуляция тикера)
- 🤖 AI-рекомендации
- 🎯 Калькулятор финансовой цели
- 🔮 Сценарный анализ на 5 лет

## Запуск

Откройте `index.html` в браузере или перейдите на GitHub Pages.
